Detailed Description

template<typename Geometry>
struct boost::geometry::traits::make< Geometry >

Traits class to create an object of Geometry type.

This trait is optional and allows to define efficient way of creating Geometries.

Geometries:
  • points
  • boxes
  • segments
Specializations should provide:
  • static const bool is_specialized = true;
  • static member function apply() taking:
    • N coordinates (points)
    • 2 points, min and max (boxes)
    • 2 points, first and second (segments)
