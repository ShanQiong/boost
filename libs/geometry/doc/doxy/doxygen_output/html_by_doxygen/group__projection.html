Initializes a projection as a string, using the format with + and =.

The projection can be initialized with a string (with the same format as the PROJ4 package) for convenient initialization from, for example, the command line

Example
srs::proj4("+proj=labrd +ellps=intl +lon_0=46d26'13.95E +lat_0=18d54S +azi=18d54 +k_0=.9995 +x_0=400000 +y_0=800000") for the Madagascar projection.
