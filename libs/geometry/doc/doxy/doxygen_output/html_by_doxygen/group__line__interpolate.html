Returns one or more points interpolated along a LineString using the specified strategy.

Template Parameters
GeometryAny type fulfilling a LineString concept
DistanceA numerical distance measure
PointlikeAny type fulfilling Point or Multipoint concept
StrategyA type fulfilling a LineInterpolatePointStrategy concept
Parameters
geometryInput geometry
max_distanceDistance threshold (in units depending on coordinate system) representing the spacing between the points
pointlikeOutput: either a Point (exactly one point will be constructed) or a MultiPoint (depending on the max_distance one or more points will be constructed)
strategyline_interpolate strategy to be used for interpolation of points
