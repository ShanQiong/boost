template<typename Geometry>
class boost::geometry::concepts::MultiPolygon< Geometry >

multi-polygon concept

Formal definition:
The multi polygon concept is defined as following:
  • there must be a specialization of traits::tag defining multi_polygon_tag as type
  • it must behave like a Boost.Range
  • its range value must fulfil the Polygon concept
